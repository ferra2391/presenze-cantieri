ON!Track Unite — API Explorer

Pagina di prova per capire come funziona l'autenticazione e i dati grezzi dell'API Hilti ON!Track Unite, prima di collegare tutto al database Python. Nessun dato lascia questo browser: le chiamate vanno dirette dal tuo dispositivo all'API Hilti.

1Autenticazione (OAuth2)

Client ID/Secret vanno in Basic Auth sull'header. Username/password del technical user vanno nel body. Il token dura 60 minuti e va rigenerato ad ogni scadenza (nessun refresh token).

2Chiamata API

Prova con /assets e /workers per iniziare (paginazione con offset/limit, default limit 50). Per lo storico posizioni di un bene/tracker, il nome esatto dell'endpoint lo trovi nello Swagger del tuo Developer Hub — una volta trovato, incollalo qui (es. /assets/{id}/transfers o simile, dato che ON!Track modella i movimenti come "transfer").
